Getting Started

My computer is a Mac, so some of this likely will not work for you, just let me know.

Your computer use a file copy of your Anki collection. First, sync your collection, and then close Anki.

On the "Your Data" page, confirm that your data has been loaded. If not, change this line:

- "/Users/hume/Library/Application Support/Anki2/Hume/collection.anki2:/hostdata/collection.anki2:ro"

In your docker-compose.yml and restart the project with this command:

docker compose up --build

Note that your terminal should be in the same folder as the docker-compose.yml

Second, there will be a card that confirms your access to the LLM.
